Add text to each post in WordPress plugin. Change text positions and content from admin panel.

== Description ==

The module delivered to your WordPress features to Add text to each post. Change text positions and content from admin
panel plugin settings.

Setting expansion

1. Text to show.
2. Where in post to show.

== Installation ==

1. Installing MaxtradeTextToPost across the dashboard for installation of plugins on WordPress or unzip files directly into
the plugins directory WordPress.
2. Upload `maxtrade-text-to-post` directory with files to the `/wp-content/plugins/` directory, select Activate plugin through
the 'Plugins' menu in WordPress.
